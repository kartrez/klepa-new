**Klepa AI** превращает вашу среду разработки в интеллектуальное рабочее пространство, где ИИ-агенты помогают не просто писать код, а понимать архитектуру проекта, проводить рефакторинг, покрывать код тестами и автоматизировать рутину.

Плагин создан с учетом **последних стандартов разработки с ИИ** (Agentic Workflow, Context-Aware), но при этом остается **максимально простым и понятным**. Низкий порог входа позволит вам начать продуктивную работу с искусственным интеллектом уже через несколько минут после установки.

## ✨ Основные возможности

*   **Автономные агенты**  
    Мощные агенты, способные самостоятельно выполнять сложные инженерные задачи под вашим контролем.
*   **Глубокая индексация кода**  
    Умный поиск и полное понимание контекста всего вашего проекта для наиболее точных ответов.
*   **Интеграция с MCP**  
    Поддержка Model Context Protocol для гибкого расширения возможностей агентов и подключения внешних инструментов.
*   **Поддержка множества LLM**  
    Свобода выбора — работайте с OpenAI, Anthropic, Google Gemini, Ollama и другими провайдерами.
*   **Безопасность данных**  
    Полный контроль над доступом. Вы сами решаете, какие файлы и директории доступны агенту.
